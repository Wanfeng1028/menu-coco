# menu-coco 🍽️

Coco 宝宝专属点餐站～

## 项目简介

一个可爱的前端点餐小网页，让她打开就能选菜、填备注、提交订单，你会收到邮件提醒。

## 技术栈

- Vue 3 + TypeScript
- Vite
- SCSS
- EmailJS
- localStorage

## 本地开发

```bash
# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入 EmailJS 配置

# 启动开发服务器
npm run dev

# 构建
npm run build
```

## 部署

推荐使用 Vercel 部署，配置环境变量后即可使用。
